# Namma Vastra

## Project Overview
Namma Vastra is an Android application developed using Kotlin and Jetpack Compose to support local weavers by digitally showcasing traditional sarees, heritage stories, and fashion trends.

## Problem Statement
Many local weavers face difficulty reaching customers digitally. This application helps weavers upload and showcase sarees online while allowing buyers to explore products easily.

## Features
- Saree Gallery
- Wishlist Functionality
- Saree Upload System
- Fashion Trend Insights
- Heritage Stories
- Price Estimator
- Firebase Firestore Integration
- Buyer and Weaver Modes

## Technologies Used
- Kotlin
- Jetpack Compose
- Firebase Firestore
- Android Studio
- Material Design 3

## Project Structure
app/
├── MainActivity.kt
├── Screens.kt
├── Firebasehelper.kt
├── Theme.kt

## Installation Steps
1. Clone the repository
2. Open the project in Android Studio
3. Sync Gradle files
4. Connect Firebase
5. Run the application on emulator or Android device

## How to Run
Open Android Studio
Click Run
Select Emulator or Device
